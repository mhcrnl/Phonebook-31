# Phonebook-31
Agenda telefonica implementata in limbajul C.

## Instalare
```
git clone https://github.com/mhcrnl/Phonebook-31.git
cd Phonebook-31
gcc main.c
```
sau deschiderea codului in Code::Blocs.

## Utilizare
```
./main
```
sau rulare in  Code::Blocks.
